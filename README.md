# README
## Ota-Chat
### :sunny:アプリ内容
- どんなアプリか<br>
趣味が同じ人同士でTwitterのような投稿をし合い、その中でグループを作って趣味が合う人同士でチャットができるようにしたアプリ<br>
- このアプリでできること<br>
  - ジャンルでの投稿のみをすることもでき、また、その中でグループを作成し、決まったメンバーとチャットすることもできます<br>
  - グループ作成では、グループにランク（ライト・ミドル・ハード）を選択でき、メンバーの趣味の度合いによって選択できるようになっています

   1. 左からグループ作成者ver,グループメンバーver,グループメンバー外verとなります<br>

   1. そのグループに所属してないユーザーは詳細の確認とそのグループに所属したい場合の希望（コメント可能）を送信することができます<br>

   1. 参加承諾のリンクと拒否のリンクがあり、グループリーダーはそのコメントなどを見て選択できるようになっている<br>
